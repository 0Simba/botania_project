    Lancement
Il faut aller sur https://fbgame.isartdigital.com/isartdigital/botaniaproject/bin/
Il y a une redirection automatique vers facebook, après avoir accepté l'application il est possible d'afficher le jeu en grande résolution en retournant sur le lien original. Le jeu prend la dimension de l'écran du navigateur.



    Boutique
La boutique ne fonctionne pas sur fbgame. Il etait possible d'afficher et d'acheter des bâtiments, mais ca ne fonctionne plus sur le serveur fbgame.
La boutique est accessible via le bouton de caddie sur le header.



    Le coffre
Plusieurs actions sont possibles via le bouton coffre du header.

Le premier onglet affiche les graines possédées par le joueur. En cliquant sur une graine, la pop up se ferme, il est possible de planter la graine.

Le second représente les fruits (aucun au départ). En cliquant dessus, une pop up pour ouvrir le fruit s'affiche. Ouvrir un fruit donne des graînes (en fonction du couteau choisi).

Le dernier représente les batiments. En cliquant dessus, la pop up se ferme et il est possible de poser le bâtiment.



    Les bâtiments
  Concasseur, batiment carré avec toit doré.
En cliquant sur un concasseur posé, un menu circulaire s'ouvre. Le bouton de gauche permet d'ouvrir le menu du concasseur.
Il faut drag n drop les graines dans les cercles puis cliquer sur "crush". Une nouvelle graîne est crée à partir des deux posées.
Un bug non fixé de pixi peut s'activer. Un refresh le corrige.

  Les bâtiments d'écosystème
Les bâtiments d'écosystème sont la ruche, l'arbre à papillon et la fourmillière.
Il est possible de les améliorer via leurs menus circulaires. Ce n'est qu'une simulation, aucune action ne sera réalisée. Cette partie n'est pas synchronisée avec le serveur



    Les plantes
Une fois une graîne plantée, elle pousse. Elle passe d'un stade toutes les une minutes. Il est possible de sauter ces étapes en arrosant la plante.
Une fois à maturité avec éclosion (animation sur la plante), il est possible de récolter son fruit en cliquant sur le bouton de droite. Le fruit apparait dans le coffre sur l'onglet fruit.



    Caméra
La caméra est contrôlable avec les flêches directionnelles.
Il est possible de zoomer et dézoomer avec les touches "a" et "q".
